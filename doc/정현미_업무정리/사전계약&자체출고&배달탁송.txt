●테이블

1) 사전계약관리
CAR_PUR_COM_PRE : 사전계약내용
CAR_PUR_COM_PRE_RES : 사전계약예약

-> 계약없어도 등록가능
-> 인기차량 선점하여 출고를 빠르게 하기위한 목적으로 시작
-> 현대특판은 기존에 사전계약만 하다가 계약입력하고 자체출고 등록이 되어야 해서, 
   사전계약등록하고, 아마존카 1단계 입력하여 자체출고까지 등록해서 처리함.
-> 출고전해지하는 경우 자체출고가 있으면 그 계출번호를 승계하는 방식으로 사전계약등록함.


2) 신차계약
CONT : 장기계약
CAR_PUR : 출고관리

-> 계약서 작성건
-> 차량대금지급요청시 출고관리 정보가 대부분 완성
-> 선급금,임시운행보험료,배달탁송 선등록하기도 함


3) 자체출고관리
CAR_PUR_COM : 자체출고영업소 출고관리
CAR_PUR_COM_CNG : 자체출고관리변경/해지내용

-> 계약입력->출고요청관리->신규계약->예정현항->배정현황->차량대금지급요청결재완료->확정현황->출고 
-> 확정전 계약변경,납품취소해지,신차취소현황으로 갈수도 있음.
-> 자체출고영업소와 업무 연동
-> 등록된 계약과 연동
-> 연동된 계약해지되면 신차취소현황으로 처리해서 대기했다가 다른계약으로 이관하기도 함(출고전해지 해지정산문서 완료시 자동처리)

car_pur_com테이블 use_yn=Y/N/C(유효,해지,변경), suc_yn=D/Y(승계대기,승계) 

출고전해지 및 신차취소현황으로 넘기로 하면 use_yn=N, suc_yn=D로 가다가, 다른 계약으로 넘어가면 suc_yn='Y'로 변경됨


4) 배달탁송
CONS_PUR : 배달탁송관리

-> 출고지에서 인수지까지 아마존카와 계약된 탁송업체를 이용하는 경우
-> 차량대금지급요청에서 등록가능함.
-> 탁송의뢰->기사배정->인수->정산



5) 출금관리
PAY_ITEM : 출금항목별세부
PAY : 출금송금

-> 차량대금, 계약금 연동 조회 등록 및 출금처리, 회계처리


6) 카드캐쉬백&상환
CARD_STAT_BASE
CARD_STAT_SCD
